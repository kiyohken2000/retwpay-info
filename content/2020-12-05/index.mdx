## 追加したコード

**src\components\Search\index.jsx**

`変更前`

```javascript
const Search = props => {
  const [focus, setFocus] = useState(false)
  const [value, setValue] = useState("")
  const onFocus = () => {
    setFocus(true)
  }
  const onBlur = () => {
    setFocus(false)
  }
  const onChange = e => {
    setValue(e.target.value)
  }
  return (
    <div>
      <TextField 
        id="outlined-search"
        label="ブログ内検索..."
        type="search"
        variant="outlined"
        onFocus={onFocus}
        onBlur={onBlur}
        onChange={onChange}
      />
      <SearchResult focus={focus} value={value} /> {/*  この行を変更します */}
    </div>
  )
}
```

**onFocus**と**onBlur**を使って**フォーカスが当たった/はずれた**の判定は前回の時点で作成済みです。なので`focus`を三項演算子で評価して**true**なら検索結果を表示するようにしました。

`変更後`

```javascript
  return (
    <div>
      <TextField 
        id="outlined-search"
        label="ブログ内検索..."
        type="search"
        variant="outlined"
        onFocus={onFocus}
        onBlur={onBlur}
        onChange={onChange}
      />
      {focus ? <SearchResult focus={focus} value={value} />: null} {/*  この行を変更した */}
    </div>
  )
```

以上です。
